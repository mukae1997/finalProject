谱面使用的jsoncpp来写入的，不做谱面就不用装了。

【】谱面记录格式】
谱面结构：
```
{ "array": 
      [   
      /*这里是一个实际谱面信息的数组*/
      ]
}
```
数组里面每一个元素都是这样的：
```
{
   "time" : 1896,  //  时间戳
   "tracks" : [    // 一个数组，包括每个轨道在这个时间戳会有什么note落下来
      {
         "length" : 1,   // note持续时间，主要是设计给长条用的
         "num" : 0,   // 轨道号码
         "type" : "tap"  // tap就是点一下
      },
      {
         "length" : 0,
         "num" : 1,
         "type" : "none" // none表示这个时刻这个轨道不会有note掉落
      },
      {
         "length" : 1,
         "num" : 2,
         "type" : "tap"
      },
      {
         "length" : 0,
         "num" : 3,
         "type" : "none"
      }
   ]
}

```
时间单位：10ms,即0.01s，因为设计为1ms的话刷新跟不上
